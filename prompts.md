# Prompts

Prompt 1 : You are a Senior AI Architect and LLM Engineer tasked with designing a production-grade Retrieval-Augmented Generation (RAG) chatbot architecture. Your goal is to design a Facts-Only Mutual Fund Assistant for Groww.com that answers factual queries about specific mutual fund schemes using only verified information from the official scheme pages provided below. The system should strictly avoid investment advice and should only return factual, source-cited responses. The chatbot is allowed to retrieve information only from the following URLs: https://groww.in/mutual-funds/quant-small-cap-fund-direct-plan-growth, https://groww.in/mutual-funds/quant-infrastructure-fund-direct-growth, https://groww.in/mutual-funds/quant-flexi-cap-fund-direct-growth, https://groww.in/mutual-funds/quant-elss-tax-saver-fund-direct-growth, https://groww.in/mutual-funds/quant-large-cap-fund-direct-growth, https://groww.in/mutual-funds/quant-esg-integration-strategy-fund-direct-growth, https://groww.in/mutual-funds/quant-mid-cap-fund-direct-growth, https://groww.in/mutual-funds/quant-multi-cap-fund-direct-growth, https://groww.in/mutual-funds/quant-aggressive-hybrid-fund-direct-growth, https://groww.in/mutual-funds/quant-focused-fund-direct-growth. These pages should be treated as the single source of truth. No additional sources should be used. Objective: Design a RAG-based chatbot architecture that answers factual questions about the above mutual funds, uses retrieval over the given links, returns concise responses with citations, prevents financial advice or return comparisons, ensures compliance and data privacy. Behavioral Rules: (1) Sources - Only use information retrieved from the provided URLs, do not use LLM pretraining knowledge, do not reference blogs/news/third-party content, do not use screenshots or internal app backend information. (2) Data Privacy - Must not collect or process PII (PAN, Aadhaar, bank account numbers, OTPs, phone numbers, email IDs, portfolio details). If users attempt to share PII, politely refuse and redirect. (3) No Investment Advice - Never provide financial advice including recommendations, fund comparisons based on returns, portfolio allocation suggestions, market predictions. If asked, respond that it cannot compute or compare returns and provide a link to the official fund page. (4) Response Format - Maximum 5 sentences, factual and neutral tone, include exact citation URL used, include "Last updated from sources: <URL>", only include the specific URL used. Task: Design a complete phase-wise architecture for building this RAG chatbot with architecture diagram. Do NOT implement code yet. Focus on architecture design only.
